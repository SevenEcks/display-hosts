Pretty print a table with the hosts file for the OS using PHP.

## Installation

Via Composer

### Global

```bash
composer global require sevenecks/display-hosts
```

### Local
```bash
composer require sevenecks/display-hosts
```

### CLI Usage

#### Global Install
```bash
DisplayHosts
```

#### Local Install
```bash
./vendor/bin/DisplayHosts
```
